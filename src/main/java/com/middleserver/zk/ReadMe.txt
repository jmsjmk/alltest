区分异步与监听的区别.
--------------------------------
监听
1.连接的时候设置监听
2.判断节点是否存在(监听,内容变更,创建当前节点,删除当前节点)
3.获取子节点列表的时候,可以注册监听--监听子节点数量的变化
4.获取子节点信息时候会监听内容变化

创建/删除/更新(没有监听)
1.同步(直接抛出异常)
2.异步(返回对应的错误码)
--------------------------------
